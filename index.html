⚙️ 구글 스프레드시트 연동 설정
📌 최초 1회 설정이 필요합니다. 아래 순서대로 진행해주세요.
설정 완료 후 모든 배차 데이터가 구글 스프레드시트에 자동 저장됩니다. (4명 이상 동시 사용 가능)
📖 설정 순서 (5분이면 끝!)
1단계. 구글 스프레드시트를 새로 만들기
2단계. 상단 메뉴 > 확장 프로그램 > Apps Script 클릭
3단계. 기존 코드를 모두 지우고, 아래 제공된 스크립트 코드를 전체 붙여넣기 후 저장(Ctrl+S)
4단계. Apps Script 상단 "배포" > "새 배포" 클릭
5단계. 유형: "웹 앱" 선택
6단계. 실행할 사용자: "나" / 액세스 권한: "모든 사용자" 선택
7단계. "배포" 클릭 → 나온 URL을 아래에 붙여넣기
8단계. 권한 승인 팝업이 뜨면 "허용" 클릭
※ 스프레드시트에서 상단 메뉴 "🚛 배차관리" > "초기 설정" 1회 실행
⚠️ 구글시트 보안 안내: 구글 스프레드시트의 공유 설정에서 "링크가 있는 모든 사용자"가 아닌 특정 사용자만 접근하도록 설정하세요. (스프레드시트 → 공유 → 일반 액세스 → 제한됨)
👥 사용자 관리 (관리자 전용)
사용자를 추가/수정/삭제할 수 있습니다. 권한별로 접근 가능한 기능이 다릅니다.
